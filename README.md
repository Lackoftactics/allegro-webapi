# Allegro::Webapi

Allegro api wrapper written in ruby.

Supported:
	-fetching user_data
	-making searches

## Installation

Add this line to your application's Gemfile:

    gem 'allegro-webapi'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install allegro-webapi
